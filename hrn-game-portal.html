<hrn-game-portal>

An all-in-one gaming hub component. Instant search, lazy-loaded tiles, and a built-in game overlay with minimalist design.

1. Setup

The portal requires the same library script as the individual game component. Include this in your <head>.

<script src="https://cdn.jsdelivr.net/gh/Odne2920/hrn-libs/1.3/lib.min.js"></script>

2. Attributes

Customize the look and data source of the entire portal using these attributes.

Attribute Default Description
theme dark Switch between dark or light color modes.
accent #4169E1 Sets the primary color for borders, search icons, and loaders.
height 600px Defines the height of the portal container (e.g., 500px or 80vh).
provider URL The base URL used to fetch game assets. Inherited by <hrn-game>.
credits true Set to "false" to hide the "Powered by" branding inside the game overlay.
card-size 220px Controls the minimum width of the game tiles in the grid.

3. Implementation Example

A clean, minimalist setup with a custom accent color and credits disabled.

<hrn-game-portal 
    theme="dark" 
    accent="#22c55e" 
    height="80vh" 
    credits="false">
</hrn-game-portal>

4. Built-in Features

The portal is designed to be autonomous and lightweight:

  • Instant Search: Filter games by name or category in real-time.
  • Minimalist UI: Clean outlines and semi-transparent "glass" backgrounds.
  • Smart Overlay: Games open in a full-container overlay without reloading the page.
  • Lazy Load: Only loads game data and assets when needed.

5. Advanced: Events

The portal emits the game:ready event from its internal <hrn-game> element, which bubbles up to the portal itself.

const portal = document.querySelector('hrn-game-portal');

portal.addEventListener('game:ready', (e) => {
    console.log("Now playing game ID:", e.detail.id);
});
