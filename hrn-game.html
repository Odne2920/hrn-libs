1. Import Script

Place this script in the <head> of your document. The component handles its own registration.

<script src="https://cdn.jsdelivr.net/gh/Odne2920/hrn-libs/lib.min.js"></script>

2. Attributes

Configure the component behavior using the following attributes.

Attribute Type Description
nr REQ Number The unique ID of the game (see the live list below).
credits Boolean Default true. Set to "false" to hide the "Powered by" branding.
redirectblock Boolean Default true. Prevents popups and top-level redirects for security.
provider URL Optional. Overrides the default game source URL.

4. Full Example

The component fills 100% of the parent container. Ensure the parent has a defined height.

<!DOCTYPE html>
<html>
<head>
    <script src="https://cdn.jsdelivr.net/gh/Odne2920/hrn-libs/lib.min.js"></script>
    <style>
        body { margin: 0; padding: 0; background: #000; height: 100vh; overflow: hidden; }
    </style>
</head>
<body>

    <div style="width: 100%; height: 100%;">
        <hrn-game nr="17" credits="false"></hrn-game>
    </div>

</body>
</html>
