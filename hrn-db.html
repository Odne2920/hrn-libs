hrn.db

The high-performance, asynchronous interface for the HyperRush Game Catalog. Includes features for searching, filtering, favorites management, and random picks.

1. Import Script

Place this script in the <head> of your document. The library automatically initializes the global window.hrn.db object. 

<script src="https://cdn.jsdelivr.net/gh/Odne2920/hrn-libs/lib2.min.js"></script>

2. Data Structure

The database consists of an array of immutable GameObject entries. All API methods return data in this format. 

interface GameObject {
  "number": 1,                           // Unique ID
  "name": "10 Minutes Till Dawn",        // Display Title
  "category": "Survival",                // Main Genre
  "img": "/images/10-minutes.webp",      // Thumbnail Path
  "link": "/games/10-minutes/",          // Game Root Path
  "device": [                            // Supported Devices
    "pc-touch",
    "pc-no-touch"
  ]
}

3. Core Methods

All methods are accessible via window.hrn.db and return a Promise (async).

.search(query)

Performs a fuzzy search across Name, Category, and Device tags. Results are sorted by relevance.

const results = await hrn.db.search("survival");

.find(criteria)

Filters the database for exact matches on specific properties.

const touchGames = await hrn.db.find({ 
    device: "pc-touch",
    category: "Survival"
});

.random()

Returns a single, randomly selected Game Object from the database.

const luckyPick = await hrn.db.random();
// Output: { name: "Bacon May Die", number: 17 ... }

.toggleFav(id) & .getFavs()

Manages user favorites via LocalStorage. No account required.

// 1. Toggle ID 17 (Add/Remove)
hrn.db.toggleFav(17);

// 2. Fetch full objects of favorite games
const myGames = await hrn.db.getFavs();

.all() & .getMeta()

Utilities for fetching the complete dataset or statistical metadata.

const allGames = await hrn.db.all();
const meta = await hrn.db.getMeta();
