hrn.stats

A "gamification" layer that sits on top of hrn-game. Automatically tracks play counts, player history, and XP progression in LocalStorage.

1. Import Script

Add this after the core library. It automatically attaches listeners to hrn-game events. 

<script src="https://cdn.jsdelivr.net/gh/Odne2920/hrn-libs/1.2/lib.min.js"></script>

2. User Stats Object

The library maintains a persistent stats object in the user's browser. You can access this via hrn.stats.get(). 

interface UserStats {
  "xp": 150,           // Total experience points (10 XP per play)
  "lvl": 3,            // User Level (Calculated from XP)
  "history": [         // IDs of last 10 played games
    17, 
    4 
  ],
  "counts": {          // Play count per ID
    "17": 5,
    "4": 1
  }
}

3. Methods

Access these methods via window.hrn.stats.

.getTop()

Returns the user's top 5 most played games as full Game Objects.

const favoriteGames = await hrn.stats.getTop();

.getHistory()

Returns the user's recently played games (last 10) as full Game Objects.

const recentGames = await hrn.stats.getHistory();

.get()

Returns the raw stats object (XP, Level, Counts).

const profile = hrn.stats.get();
console.log("Current Level:", profile.lvl);

.reset()

Wipes all progress and history for the user.

hrn.stats.reset();
